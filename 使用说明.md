# 语音输入工具 — 使用说明

中文语音转文字。打开浏览器页面后，点按钮录音，识别完点“复制”，再到任何地方 Ctrl+V 粘贴。

本发布包不包含任何 key。首次运行 `setup.bat` 会从 `.env.example` 创建 `.env`，请先打开 `.env` 填入你的 AI Gateway key 或豆包官方 key。不要把 `.env` 发到公开群或提交到 git。

---

## 第一次使用（只做一次）

### 1. 装 Python

到 [https://www.python.org/downloads/](https://www.python.org/downloads/) 下载 **Python 3.10 或更新版本**。

> **重要**：安装界面第一页 **务必勾选** `Add Python to PATH`，否则后面会失败。

### 2. 解压本压缩包

随便解压到桌面或哪都行。

### 3. 双击 `setup.bat`

会弹出黑色命令行窗口，自动装好需要的东西，约 1-2 分钟。
看到 `安装完成！` 关掉就行。

---

## 日常使用

**双击 `start.bat`** 即可。浏览器会自动打开。

如果还没填 `.env`，页面能打开，但录音结束后识别会报缺少 key。

操作很简单：

1. 点页面上的 **「开始录音」** 按钮
2. 对着麦克风说话（中文）
3. 再点一次按钮（变成「结束录音」）结束
4. 识别结果出现在页面，自动加进下方历史
5. 历史里点 **「复制」** 按钮 → 到要用的地方 Ctrl+V

默认是浏览器按钮版：不绑定全局热键，不自动粘贴。

**历史记录**

- 可以**编辑**（点编辑按钮 / 按 Enter 保存）
- 可以**复制**或**删除**
- 关掉浏览器还在，不丢

**退出**：关掉黑色命令行窗口就停了。

## 常见问题

**Q：浏览器打不开 / 提示无法连接**
A：等 5 秒再刷新。第一次启动稍慢。

**Q：点了"开始录音"没反应？**
A：浏览器会问"允许使用麦克风"——点允许。如果之前误点了禁止，到浏览器地址栏左侧锁形图标里改回允许。

**Q：识别不准 / 没声音**
A：检查 Windows 系统的麦克风音量，确认录的是你想用的麦克风（系统设置 → 声音 → 输入）。

**Q：专名经常错？**
A：打开 `voice_input/hotwords.txt`，把你常用的人名、项目名、英文缩写一行一个加进去。

**Q：电脑重启后想用？**
A：双击 `start.bat` 即可，不需要重新 `setup`。

**Q：Windows 防火墙弹窗？**
A：选"允许"。本程序只在本机用，不连外网（除了识别时连豆包语音 API）。

## 技术细节（可忽略）

- 端口：`http://127.0.0.1:8770/`
- 历史存在：`%APPDATA%\VoiceInput\history.sqlite3`
- 识别用：AI Gateway `/audio/transcriptions`
- 关闭：直接关命令行窗口
